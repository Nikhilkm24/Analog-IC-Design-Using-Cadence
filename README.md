# Analog IC Design Using Cadence

This repository documents my **day-wise hands-on learning** from an
**Analog IC Design workshop** using **Cadence Virtuoso**.
The focus is on understanding **analog circuit behavior**, simulation
results, and practical design insights rather than just tool usage.

---

##  Objectives
- Build strong fundamentals in Analog IC Design
- Understand MOS device operation through simulations
- Analyze amplifier behavior using AC, DC, and transient analysis
- Design and evaluate Bandgap Reference (BGR) circuits
- Learn industry-style documentation and debugging practices

---

##    Tools & Environment
- Cadence Virtuoso (ADE L / XL)
- Spectre Simulator
- Linux-based Design Environment

---

##  Topics Covered
- MOS Device Fundamentals
- Common-Source Amplifiers
- Current Mirrors
- Frequency Response & Stability
- Bandgap Reference Circuits
- AC, DC, and Transient Analysis
- Practical Debugging in Cadence
